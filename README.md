# "树影拼贴生成器" (Tree Silhouette Collage Generator) 功能说明

本方案采用单文件（HTML + CSS + JavaScript）架构，利用浏览器原生性能实现复杂的图像处理和动态艺术生成。

---

## 1. 核心功能结构

### **A. 智能图像处理模块 (Image Engine)**
*   **双源输入**：
    *   **主图**：用户上传树木剪影照片。
    *   **素材池**：允许上传多张图片（花朵、自然纹理等），存储于内存数组。
*   **动态树枝提取算法**：
    *   **边缘采样**：自动获取图像四周边缘像素，计算平均亮度作为背景参考基准。
    *   **灵敏度控制**：实时二值化处理。用户调节滑块时，Canvas 会以红色高亮显示识别出的树枝区域（洋葱皮预览模式）。
    *   **Uint8Array 遮罩**：使用位数组存储树枝与背景区域，确保每秒 60 帧的实时处理性能。
    *   **边缘复原 (Edge Restoration)**：在生成结束后，根据遮罩重新覆盖原始树枝像素，确保拼贴边缘与树木轮廓完美契合，无溢出或锯齿。

### **B. 艺术生成模块 (Art Generator)**
*   **自然连通域识别 (Connected Component Labeling)**：
    *   **算法逻辑**：系统自动寻找由树枝剪影与画布边缘共同围成的"闭合背景区域"。
    *   **形态学侵蚀 (Erosion) 融合**：当增加"最小区块大小"时，算法会同步对树枝遮罩进行像素级侵蚀。这会导致细小枝干在算法层面"消失"，从而让原本被分割的背景孔洞自动合并成更大的艺术区块。
    *   **区块定义**：每一个被树枝分割出来的"孔洞"或"背景色块"将作为一个独立的拼贴单元，不再进行人为的网格划分。
    *   **噪点过滤**：支持通过"最小区块大小"滑块过滤微小的碎屑区域，确保视觉纯净度。
*   **双模式填充系统**：
    *   **随机填充模式**：所有区块自动从素材库中随机选择图片进行填充。
    *   **手动拖拽模式**：
        *   **数字区块系统**：为每个识别出的区块分配编号（1、2、3...），在素材库上方显示对应的数字块。
        *   **拖拽填充**：用户可以将素材库中的图片拖拽到对应的数字块中，实现精确控制。
        *   **实时预览**：拖拽后立即在成果展示区显示填充效果。
        *   **智能高亮**：已填充的区块显示绿色边框，未填充的区块显示橙色边框。
        *   **自适应文本颜色**：根据背景颜色自动调整数字文本颜色，确保在深色和浅色背景下都清晰可见。
*   **无缝填充策略**：
    *   **像素级蒙版**：为每个连通域生成精确的像素蒙版，利用 `source-in` 混合模式将素材灌入。
    *   **智能旋转 (Adaptive Orientation)**：利用图像矩 (Image Moments) 算法计算每个封闭区块的长轴角度。开启后，素材图片会自动旋转至契合孔洞延伸方向，增强视觉流向感。
    *   **立体阴影 (Drop Shadows)**：支持为每个拼贴区块开启实时投影效果，营造出剪纸或拼贴画的层叠深度感。
    *   **自定义混合模式**：提供多种 Canvas 混合模式（正常、正片叠底、滤色、叠加、柔光），让素材与背景产生更丰富的艺术融合。
    *   **纹理映射**：素材图片经过 `Object-Fit: Cover` 逻辑计算，确保完全覆盖区块且不留白。
*   **填充顺序存储机制**：
    *   **自动存储**：每次生成时自动保存每个区块使用的素材ID和顺序。
    *   **录制复用**：录制视频时使用已保存的填充顺序，确保视频与生成效果一致。
    *   **模式隔离**：切换填充模式时自动清空填充状态，但保留主图和素材库。
*   **动态填充动画**：
    *   **从根到梢 (Root-to-Tip)**：按区块底部坐标排序，实现逐个"生长式"填充效果。
    *   **节奏控制**：
        *   **生成模式**：使用 0.001 秒超快间隔，几乎瞬间完成生成。
        *   **录制模式**：使用用户设置的间隔时间（0-5秒，可手动调整），控制视频中的填充速度。
    *   **淡入效果**：每个区块都有 1000ms 的淡入动画，增强视觉流畅度。
    *   **视频导出优化**：在动画结束以后增加 5 秒的静止展示时间。为了解决部分浏览器在静态画面下停止录帧导致时长缩短的问题，程序会在静止期持续推送 Canvas 刷新信号。同时支持自动检测浏览器环境，优先尝试导出 MP4 格式（如 Safari），否则回退至 WebM 格式。
    *   **实时区块预览**：在调节"最小区块大小"或"灵敏度"时，可切换至"区块分割预览"模式，通过多色块实时展示识别到的自然孔洞。

### **C. 视频捕捉与输出 (Output Module)**
*   **高清下载**：支持将 Canvas 内容转换为 4K 级别的 PNG 格式。
*   **视频生成**：利用 `MediaRecorder` 录制 Canvas 动画流，生成 WebM/MP4 格式视频。

## 2. 界面设计布局 (UI/UX)

界面采用现代感十足的**半透明毛玻璃风格**，分为三个核心区域：

| 区域 | 功能组件 |
| :--- | :--- |
| **左侧控制面板** | 主图上传、灵敏度滑块、分割块数量调节、混合模式选择、艺术增强选项、填充模式选择、录制间隔设置、背景样式切换、生成/导出/录制按钮。 |
| **中间预览区** | 遮罩预览、区块分割预览、成果展示、状态信息和进度条。 |
| **底部素材栏** | 数字块显示区（手动模式）、素材上传、素材缩略图展示。 |

### **界面特色功能**

1. **混合模式选择器**：
    *   圆形图标样式，直观展示不同混合模式的效果。
    *   支持 5 种混合模式：正常(N)、正片叠底(M)、滤色(S)、叠加(O)、柔光(L)。

2. **填充模式切换**：
    *   随机填充：全自动随机生成。
    *   手动拖拽：数字块系统，支持精确控制。

3. **背景样式选择**：
    *   原始主图、透明、纯黑、纯白、午夜蓝、朱红、翡翠绿、向日葵黄、紫罗兰、湿沥青等多种预设。

4. **录制间隔控制**：
    *   数字输入框，支持 0-5 秒范围，精确到 0.01 秒。
    *   只影响录制视频，不影响生成速度。

5. **数字块系统**：
    *   手动模式下显示，随机模式下隐藏。
    *   显示区块编号和填充状态。
    *   支持拖拽素材填充。

## 3. 技术执行流程

### **A. 初始化与遮罩生成**
1.  **读取主图**，计算灰度：$L = 0.299R + 0.587G + 0.114B$。
2.  **应用阈值** $T$（受滑块控制），标记"树枝区"与"填充区"。
3.  **生成遮罩**：使用 Uint8Array 存储二值化结果。

### **B. 空间分割**
1.  **连通域检测 (CCL)**：识别所有被遮罩包围的独立背景区域。
2.  **应用"最小区块过滤"**：剔除孤立噪点。
3.  **形态学侵蚀**：根据"最小区块大小"参数对遮罩进行侵蚀，合并小区域。

### **C. 填充与录制**
#### **随机模式流程**
1.  **检测模式**：判断当前是否为随机填充模式。
2.  **随机选择素材**：为每个区块随机选择素材库中的图片。
3.  **预渲染**：为每个区块预渲染素材，提高动画性能。
4.  **存储顺序**：将每个区块的素材ID和索引保存到 `savedFillOrder` 数组。
5.  **动画生成**：使用 0.001 秒间隔快速生成动画。
6.  **录制准备**：录制时使用已保存的填充顺序。

#### **手动模式流程**
1.  **检测模式**：判断当前是否为手动拖拽模式。
2.  **检查填充状态**：遍历所有区块，检查 `filledRegions` 状态。
3.  **保留已填充**：对于已填充的区块，使用用户拖拽的素材。
4.  **随机填充空块**：对于未填充的区块，从素材库中随机选择。
5.  **预渲染**：为每个区块预渲染素材。
6.  **存储顺序**：将每个区块的素材ID和索引保存到 `savedFillOrder` 数组。
7.  **动画生成**：使用 0.001 秒间隔快速生成动画。
8.  **录制准备**：录制时使用已保存的填充顺序。

#### **模式切换处理**
1.  **清空状态**：切换填充模式时清空 `filledRegions`、`manualRegions`、`savedFillOrder`。
2.  **清空图片**：清空数字块中的图片，但保留数字块显示。
3.  **保留资源**：保留主图和素材库，不重新上传。
4.  **更新预览**：刷新预览区域。

#### **录制流程**
1.  **检查填充顺序**：判断是否有已保存的填充顺序。
2.  **使用保存顺序**：如果有，使用 `savedFillOrder` 中的素材和顺序。
3.  **设置间隔**：使用用户设置的录制间隔时间。
4.  **开始录制**：启动 MediaRecorder。
5.  **动画展示**：按保存的顺序和间隔播放动画。
6.  **静止展示**：动画结束后保持 5 秒静止。
7.  **停止录制**：生成视频文件并自动下载。

### **D. 视频录制**
1.  **开启 `MediaRecorder`**。
2.  **循环执行** `clip()` 和 `drawImage()`，按排序触发动画。
3.  **动画结束**，生成下载链接。

## 4. 性能与优化策略

*   **离屏渲染**：高清导出时使用 `OffscreenCanvas`，防止界面卡顿。
*   **素材预处理**：上传素材自动缩放至 1024px 左右，平衡画质与显存占用。
*   **预渲染优化**：为每个区块预渲染素材，避免动画过程中重复处理。
*   **填充顺序缓存**：保存填充顺序，避免录制时重新计算。
*   **模式状态隔离**：不同模式之间的状态独立管理，避免相互干扰。
*   **快速生成**：生成时使用 0.001 秒间隔，几乎瞬间完成。
*   **录制间隔控制**：录制时使用用户设置的间隔，提供更好的视觉效果。

## 5. 使用指南

### **基本操作流程**
1.  **上传主图**：点击"选择图片"按钮，上传树木剪影照片。
2.  **上传素材**：点击底部素材栏的"+ 添加素材"按钮，上传多张素材图片。
3.  **调整参数**：
    *   调整"识别灵敏度"滑块，控制树枝识别的精度。
    *   调整"最小区块大小"滑块，过滤小区域。
    *   选择"混合模式"，控制素材与背景的融合效果。
    *   开启/关闭"智能旋转"和"立体阴影"。
4.  **选择填充模式**：
    *   **随机填充**：自动随机生成。
    *   **手动拖拽**：使用数字块系统精确控制。
5.  **设置录制间隔**：输入 0-5 秒之间的数值，控制录制视频时的填充速度。
6.  **选择背景样式**：选择合适的背景颜色。
7.  **生成图片**：点击"生成"按钮，生成最终拼贴图片。
8.  **导出图片**：点击"导出"按钮，下载 4K PNG 图片。
9.  **录制视频**：点击"录制"按钮，生成 WebM/MP4 视频。

### **手动拖拽模式详细操作**
1.  切换到"手动拖拽"模式。
2.  系统自动识别区块并显示数字块。
3.  将素材库中的图片拖拽到对应的数字块中。
4.  已填充的区块显示绿色边框。
5.  未填充的区块显示橙色边框。
6.  点击"生成"按钮，已填充的区块保留素材，未填充的区块随机生成。
7.  点击"录制"按钮，使用当前的填充顺序录制视频。

### **模式切换说明**
*   切换填充模式时，数字块中的图片会被清空，但数字块（编号）会保留显示。
*   主图和素材库不会受影响，不需要重新上传。
*   填充状态会被重置，需要重新设置。

## 6. 技术特性

### **核心算法**
*   **连通域标记 (Connected Component Labeling)**：识别独立区域。
*   **图像矩计算 (Image Moments)**：计算区域角度，实现智能旋转。
*   **形态学操作 (Morphological Operations)**：侵蚀和膨胀，优化区域分割。
*   **混合模式 (Blend Modes)**：Canvas globalCompositeOperation 实现。

### **性能优化**
*   **Uint8Array 遮罩**：高效的像素级操作。
*   **预渲染缓存**：减少重复计算。
*   **requestAnimationFrame**：流畅的动画渲染。
*   **MediaRecorder**：原生视频录制支持。

### **浏览器兼容性**
*   **自动格式检测**：优先尝试 MP4，回退至 WebM。
*   **Canvas 2D API**：广泛支持的图形渲染。
*   **FileReader API**：本地文件读取。
*   **拖拽 API**：原生拖拽支持。

## 8. 注意事项

1.  **素材要求**：建议上传高分辨率、色彩丰富的素材图片。
2.  **主图要求**：建议使用对比度高的树木剪影图片。
3.  **录制间隔**：间隔时间越长，视频文件越大。
4.  **浏览器兼容性**：推荐使用最新版本的 Chrome、Firefox、Safari 或 Edge 浏览器。
5.  **性能建议**：区块数量过多时，生成和录制时间会相应增加。
6.  **模式切换**：切换填充模式时会清空填充状态，请注意保存重要作品。

## 9. 故障排除

### **常见问题**
*   **录制视频无法播放**：尝试更换浏览器或调整录制间隔。
*   **区块识别不准确**：调整"识别灵敏度"和"最小区块大小"滑块。
*   **生成速度慢**：检查素材数量和区块数量。
*   **手动拖拽不显示**：确保切换到手动模式，并刷新页面。
*   **录制视频与生成不一致**：确保先点击生成，再点击录制。

### **性能优化建议**
*   **减少区块数量**：增加"最小区块大小"参数。
*   **优化素材大小**：使用适当大小的素材图片。
*   **关闭高级效果**：暂时关闭"智能旋转"和"立体阴影"。
*   **使用现代浏览器**：确保最佳性能和兼容性。
